## Getting Started

### Clone-Locally

- `git clone https://github.com/abel-blue/yt-clone`
- `npm i -g npm-check-updates`
- `ncu -u`
- `npm install`

### Prerequisites
* npm
  ```sh
  npm install npm@latest -g
  ```
